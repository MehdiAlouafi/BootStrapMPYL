About the Course

One Month Responsive Design is the next best step for taking your front-end web development skills from the beginner to intermediate level. In this course, Chris Castiglione (One Month Instructor and Dean) will teach you to write better front-end code so that you can “Code it once, and display it anywhere.”

iPhone, iPad, laptop, Android, Chrome, Safari, full-screen, narrow width… the truth is that you have no idea where your Website is being viewed. And without the proper code, it’s likely to display incorrectly - resulting in some unhappy users. In this course, Chris will help you tackle those tough questions of HTML & CSS best-practices and code optimization so that you can build something awesome that lasts forever! (...or at the very least a really, really long time).

This course requires a basic working knowledge of HTML and CSS. Need a refresher? Check out Chris’s other course: One Month HTML!
